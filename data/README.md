# Data

PDF papers go here. They are gitignored so you don't accidentally commit copyrighted material.

## Suggested sources

- **arXiv** — open-access. Useful queries: `autonomous driving`, `end-to-end driving`, `behavioral cloning`, `lane keeping`.
  - https://arxiv.org/list/cs.RO/recent
  - https://arxiv.org/list/cs.CV/recent
- **NHTSA** — public US safety reports.
- **EU CARS 2030** — European public reports on automotive AI.
- **OpenReview** — ICLR/NeurIPS papers (free PDFs).

For the demo, 5–10 papers is enough. After dropping PDFs in this folder, run:

```bash
python -m rag.ingest
```
